# PL/SQL Payroll Management System

## Overview
A self-practice PL/SQL project that demonstrates implementing payroll business logic
at the database layer using SQL and PL/SQL concepts.

## Features
- Relational tables for employees, attendance, salary, and audit logs
- PL/SQL stored procedure for monthly payroll calculation
- Cursor-based processing with exception handling
- Trigger-based audit logging
- Tested with realistic sample employee and attendance data

## Tech Stack
- SQL
- PL/SQL 
- Oracle Database XE
- Oracle SQL Developer

## How to Run
1. Connect to Oracle Database using SQL Developer
2. Run `run.sql`
3. Verify results:
   - `SELECT * FROM salary;`
   - `SELECT * FROM audit_log;`

## Learning Outcome
Hands-on experience with PL/SQL procedures, cursors, triggers, anddatabase-driven business logic implementation.
